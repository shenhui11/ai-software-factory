- 在现有MVP的基础上，还需要加上全自动的流程； 
- 全自动流程说明：
  - 每章先由故事编辑 Agent 产出多个走向方案（3个） ， 另外一个AI agent （资深主编）进行打分- 情节推动、故事一致性、是否足够吸引眼球，选择分数最高的 继续进行；
  - 然后由写手 Agent 生成正文 ，资深读者 Agent 打分并给出问题说明（最多回流5次，选择评分最高的章节）
  - 每次最高支持续写10章内容；
